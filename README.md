# michael.miousse

Static bilingual (English/French) website to promote Michael Miousse books.

## Pages

- `/` language selector
- `/en/` English home
- `/en/book.html` English book page
- `/en/about.html` English author page
- `/fr/` Accueil français
- `/fr/livre.html` Page livre française
- `/fr/a-propos.html` Page auteur française
